# Paper Monitor 0.1.7

This release expands the journal catalog and makes large journal selections practical in the Windows app.

## Windows downloads

- `Paper-Monitor-Windows-0.1.7-Setup.exe`: normal per-user installer.
- `Paper-Monitor-Windows-0.1.7.zip`: portable application bundle.
- `Paper-Monitor-Windows-0.1.7.exe`: standalone executable.
- `SHA256SUMS-0.1.7.txt`: SHA-256 checksums for all Windows artifacts.

## Highlights

- 300 formal journals across 12 interdisciplinary categories, plus optional arXiv.
- Search and category filtering in Settings and Keyword Analysis.
- Compact single-line journal rows with frozen OpenAlex `2Y Impact` values.
- Fixed journal removal and explicit empty-selection persistence.
- Bounded internal list scrolling and responsive desktop/narrow-window layouts.

## Metric note

`2Y Impact` is the OpenAlex two-year mean citedness snapshot retrieved on 2026-07-10. It is an open citation-impact indicator and is not Clarivate Journal Impact Factor.

Windows binaries are unsigned unless a trusted Authenticode certificate is configured in the release workflow.
